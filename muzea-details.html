  2. Muzea e-commerce Details

Project information

  • Category: Web e-commerce
  • Context: Final school project
  • Project date: September 2023 - October 2023
  • GitHub: github.com/HasretMengi/muzea-magasin
  • Deployed Link: muzea-magasin.vercel.app/
  • Used technologies: React, Next.js, NextAuth.js, Tailwind.css, styledcomponents, MongoDB, AWS, Stripe, DialogFlow, Vercel, Google Cloud Platforms.

Features

  • Home Page that has a showcase art piece and the latest additions
  • Description and reviews from clients for each art piece
  • Categories with filters according to their respective properties and common filters like price and added date
  • Account creation with Google to have access to a wishlist and previous orders
  • Chatbot that assists clients with their requests

Possible improvements

  • Create a client reward system
  • Better trained chatbot
  • Change colors to have a more modern look
